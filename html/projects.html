Pcap Tracker

GitHub Repo

The goal of this project was to create a service that analyzes network traffic from different devices. Initially inspired by a curiosity about smart home device traffic, it evolved into a tool that visualizes network activity for any device of interest. Users can upload a PCAP (packet capture) file and specify an IP or MAC address to analyze, then view visualizations of traffic volume over time, communication partners, and geographic destinations. The application is live at https://pcaptracker.site, or users can run it locally via Docker by cloning the GitHub repo.

Key Features:

  • Developed custom Python scripts using the Scapy library for packet analysis
  • Utilized multithreading to optimize PCAP processing, ensuring efficient large file analysis
  • Created interactive data visualizations using D3.js to display network traffic insights
  • Built a responsive front-end using HTML, CSS, and JavaScript, hosted on a Linux cloud server
  • Implemented a Flask middleware, served with Gunicorn, to handle requests between the frontend and backend
  • Configured Apache as a reverse proxy to Flask, ensuring secure and scalable deployment
  • Packaged the application with Docker for simplified deployment and local execution

Few-Shot ML Pneumonia Classifier

GitHub Repo

A major challenge in medical machine learning is acquiring large, high-quality datasets for training models. To address this, my team and I developed a few-shot learning model that requires significantly fewer images to train while maintaining high accuracy. Our model needed only ~50 images to achieve ~88% accuracy, compared to conventional models requiring thousands.

Key Features:

  • Utilized common Python ML libraries including NumPy, TensorFlow, Keras, and Pandas
  • Built a ResNet-based ML model optimized for low-data scenarios
  • Achieved 98% of the performance of a standard model trained on ~3000 images

Social Compass Android App

GitHub Repo

Developed an Android application in a team of six using Agile development practices. The app allows users to add friends via unique codes and track their real-time locations on a dynamic compass UI. Friend locations update continuously as they move throughout the day, with data synchronized via a cloud-hosted AWS SimpleDB server.

Key Features:

  • Built with Java in Android Studio, utilizing Gradle for consistent builds
  • Implemented real-time location tracking with live updates from a cloud-hosted database
  • Designed a compass-based UI that dynamically rotates based on user movement
  • Added zoom functionality to adjust the view based on friend proximity
  • Ensured continuous background location updates for seamless tracking
  • Implemented a CI/CD pipeline for automated testing on code pushed
